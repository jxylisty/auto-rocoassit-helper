# 洛克王国异色繁育规划器

## 运行说明

由于使用了 Web Worker 技术，**不能直接双击打开 HTML 文件**，需要通过本地服务器来运行。

### 方法一：使用 Python（推荐）

如果你安装了 Python，双击运行 `start-server.bat`，然后在浏览器中打开：
```
http://localhost:8000/roco_shiny_breeding_planner.html
```

### 方法二：使用 PowerShell

右键点击 `start-server.ps1`，选择"使用 PowerShell 运行"，然后在浏览器中打开：
```
http://localhost:8000/roco_shiny_breeding_planner.html
```

### 方法三：使用 VS Code Live Server

如果你使用 VS Code，安装 "Live Server" 扩展，右键点击 `roco_shiny_breeding_planner.html` 选择 "Open with Live Server"。

## 功能说明

- 📊 输入异色公母库存
- 🧮 智能计算最优摆窝方案
- 📈 可视化布局展示
- 📋 详细配对列表
